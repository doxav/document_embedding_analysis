Data ingestion is the process of getting data streams from its source to its processing or storage system as efficiently and correctly as possible  . The ingestion layer ensures scalable, resilient, and fault-tolerant data distribution across the DSPS architecture from multiple input data streams, and decouples the input data sources from the other parts of the DSPS  ,  . These input data streams can be comma or tab delimited text, graphs, JSON objects, WebSockets, events or any timeseries data. The data stream ingestion layer deploys a variety of tools to accept different types of input data streams from one or more sources.
