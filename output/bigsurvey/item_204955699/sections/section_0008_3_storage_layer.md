DSPSs often store analyzed data, discovered patterns and extracted knowledge from different data processing stages for further processing BIB38200299 . Stored data must be organized and indexed along with external knowledge or metadata for executing subsequent analytics jobs. Data storage solutions for supporting DSPS architecture ranges from traditional file systems such as HDFS and Baidu File System (BFS) to distributed file relational databases such as PostgreSQL, key-value stores such as Redis, in-memory databases such as VoltDB, document storage such as MongoDB, graph storage systems such as Neo4j, NoSQL databases such as Cassandra, and NewSQL such as Cock-roachDB BIB5054652 .
