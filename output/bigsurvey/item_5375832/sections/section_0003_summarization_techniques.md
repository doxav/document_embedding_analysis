Summarization techniques can be classified according to the factors presented in section 2.1. For example, they can be classified according to the number of input documents (single-document vs. multi-document), to the type of these documents (textual vs. multimedia), to the output types (extractive vs. abstractive), etc. In this section, the various summarization techniques are presented under the following classification: • Extractive • Abstractive • Multidocument
