# BigSurvey MQS Output

This directory is generated by:

```bash
python scripts/import_bigsurvey.py --n 20 --split test --skip-embeddings
```

Generated files include `dataset_manifest.json`, `MQS_evaluation_dataset.jsonl`, and
`item_*` folders. They are ignored by git because they are reproducible from the
cache in `data/bigsurvey/`.
