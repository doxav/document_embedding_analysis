In some scenarios, it is reasonable to estimate multiple source reliabilities for a single source, and thus the variety in source reliability can be captured. The authors in  demonstrate that when objects can be clustered into sets, it is better to estimate a source reliability degree for each object set. Similarly, as a website may have very different reliabilities for different attributes of objects, the authors in BIB2058597 treat a website as multiple sources according to attributes and objects, instead of treating the website as a single source. This is equivalent to assigning multiple source reliability degrees to a single website. In BIB6916259 , the authors propose a probabilistic graphical model to jointly learn the latent topics of questions (objects), the fine-grained source reliability, and the answers to questions (truths). Furthermore, as the number of provided observations becomes very large, the source consistency assumption may not hold any more, and it is more reasonable to have multiple source reliabilities per source.
