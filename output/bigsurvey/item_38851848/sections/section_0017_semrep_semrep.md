There are other text processing tools that have been used in the past for medical text processing although their functionalities are not restricted to this domain. The following is a brief list: -GATE 31 : It is a Java platform from the University of Sheffield, and, according to their website, it is capable of solving almost any text processing problem. -LingPipe 32 : A useful, Java-based language processing tool that is free for non-commercial use and is widely used for research. -OpenNLP 33 : It is a source for a variety of Java-based NLP tools that can perform a range of basic and advanced text processing tasks.
